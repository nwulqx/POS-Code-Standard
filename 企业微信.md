对于 drs-wecom-pos 项目中，使用 figma 进行开发时，需要注意：

1. 项目默认 2 倍 px 的指定，比如 figma 看到 10px，则使用 css 时需要写为 20px，对于所有 px 设置都是如此
2. 项目默认字体 font-size 是 28px(对应 figma 显示为 14px)，如果设计稿对于字体大小定义是 14px，则不需要指定大小；否则则需要
3. 对于 font-family 因为也全局默认了，则也不需要进行声明定义了
4. css命名规则统一使用 小写+横杠 的表达方式，不要用下划线，驼峰方式定义css，其他除外
5. 非必要不要加注释，好的代码通过阅读即可理解，如果是复杂逻辑，请从输入输出来解读，并从宏观能力上添加注释，代码逻辑不必给每一步都加，这时冗余+维护成本。
